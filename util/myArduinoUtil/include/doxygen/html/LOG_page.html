myLOG

Programm for logging

Description

Currently, the logging programm is able to handle the levels error, info and debug. The idea is to cover all log levels such as:

  • ALL = 1
  • DEBUG = 2
  • INFO = 3
  • WARN = 4
  • ERROR = 5
  • FATAL = 6
  • OFF = 7

The log level is defined by setting the Variable GLOBAL_LOG_LEVEL to the metioned integer value.
